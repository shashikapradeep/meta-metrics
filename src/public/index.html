Metrics reference & overlap guide

Organic Instagram Graph API

impressions
Removed in Instagram API v22.0+. No longer available for organic posts — only paid impressions are returned by the Ads Insights API.
reach
Unique accounts that saw the post at least once.
likes
Tap-heart reactions on the organic post.
comments
Comments left directly on the organic post.
shares
Times the post was shared via DM or story.
saves
Times the post was bookmarked.
plays / views
Video starts (Reels / Video only).

Paid Meta Ads Insights API

impressions
Times the ad was shown. Organic impressions are unavailable (removed in API v22.0+), so this is the only impressions figure in the response.
reach
Unique people reached by the ad. Overlaps with organic reach — the same person can appear in both counts when the post was also distributed organically.
clicks
Any click on the ad (link, profile, photo expand). Not available in organic metrics.
spend
Total amount charged for the boost in the account currency. Not available in organic metrics.
actions › link_click
Clicks specifically on a link in the ad (subset of clicks).
actions › post_engagement
All interactions rolled up: likes, comments, shares, saves, reactions, video views. Overlaps with all organic engagement counters.
actions › page_engagement
Interactions that also count toward the connected Facebook Page (supersets post_engagement in some placements).
actions › post_reaction
Emoji reactions (Like, Love, Haha, etc.). Overlaps with organic likes — Instagram counts these as likes.
actions › post
Shares of the ad post to a user's own feed or story.
actions › video_view
3-second+ video views via the ad. Overlaps with organic plays/views for boosted videos.
actions › onsite_conversion.post_save
Saves via the ad. Overlaps with organic saves.
actions › onsite_conversion.post_net_like
Net new likes gained through the ad (saves minus unlikes).
actions › onsite_conversion.post_unlike
Unlikes that happened after the ad-driven like.
actions › post_interaction_gross
Raw interaction count before deduplication (sum of all action types, may double-count).

How Instagram calculates the counts shown on a post

When a post is boosted, Instagram combines organic engagement with ad-driven engagement into a single number. The formula for each visible counter is:

Shown on post Organic field + Paid action field Note
LIKES organic.likes onsite_conversion.post_net_like Net likes = new likes minus unlikes from the ad audience.
COMMENTS organic.comments comment Direct comments left via the ad placement.
SHARES organic.shares post Re-shares of the post to feed or story via the ad.
SAVES organic.saved onsite_conversion.post_net_save Net saves = bookmarks minus unsaves from the ad audience.

Example — post DVuIhDoAchU:
Likes: 1,527 organic + 9,247 paid net likes = 10,774 → shown as 10.7K
Comments: 10,473 organic + 15 paid comments = 10,488 → shown as 10.5K

Why the API returns separate numbers instead of the combined total: Organic metrics come from the Instagram Graph API (media insights) and paid metrics come from the Meta Ads Insights API. These are two independent systems — each records engagement from its own distribution channel. Instagram's app UI deduplicates and merges them; this API response keeps them separate so you can see the contribution of each channel.
